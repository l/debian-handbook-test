1.4. 跟著 Debian 最新消息

如前所述，Debian 計畫以極為分散、有機的方式進行。結果是，必須痛苦地讀取無止盡的訊息才能保持與計畫連繫。
訂閱 debian-announce@lists.debian.org 郵寄名單是獲知 Debian 重要訊息的方法。此郵寄名單的流量極低 (每年祗有十來個訊息)，祗提供最重要的通知，諸如新版釋出消息、選舉新的計畫領導人、或 Debian 年度會議。
→ https://lists.debian.org/debian-announce/
有關 Debian 的一般 (與定期) 消息經由 debian-news@lists.debian.org 郵寄名單發送。其流量尚可 (通常每月數則)，包括半定期的 'Debian 計畫消息'，匯編計畫內發生的小事。Debian 發展者在這裡發布有必要讓大眾知悉的事，算是內幕消息。
→ https://lists.debian.org/debian-news/

社群 公關與媒體團隊

Debian 的官方溝通管道由公關團隊與媒體團隊的志工擔任。媒體團隊是 Debian 計畫領導人的代表釋出官方的消息。公關團隊較不正式且歡迎大家參與，提供稿件給 'Debian 計畫消息' 或供稿給@debian Identi.ca 微部落客。
→ http://wiki.debian.org/Teams/Press
→ http://wiki.debian.org/Teams/Publicity
Debian 的演化與各團隊的運作，參見 debian-devel-announce@lists.debian.org 郵寄名單。如其名稱所示，其內容較適合發展者的口味，也適合關心穩定版以外版本發展的人。debian-announce@lists.debian.org 提供使用者可見的結果，debian-devel-announce@lists.debian.org 則告知該等結果的產生方式。如專欄所示，'d-d-a' (如其名所示) 是 Debian 發展者必須訂閱的郵寄名單。
→ https://lists.debian.org/debian-devel-announce/
詳盡的非正式資訊在 Debian 星球，匯集 Debian 奉獻者在自身部落格的文章。其內容不以 Debian 發展為限，還包括社群的活動與會員的動態。
→ http://planet.debian.org/
計畫本身也是一種社群網路。Debian 本身祗有以自由軟體建構的官方平台 (如 Identi.ca 微部落格平台，由 pump.io) 架構，Debian 的奉獻者多半另有自己的推特帳號、臉書頁面、Google+ 頁面等。
→ https://identi.ca/debian
→ https://twitter.com/debian
→ https://www.facebook.com/debian
→ https://plus.google.com/111711190057359692089
