1.4. 跟着 Debian 最新消息

如前所述，Debian 计划以极为分散、有机的方式进行。结果是，必须痛苦地读取无止尽的消息才能保持与计划连系。
订阅 debian-announce@lists.debian.org 邮寄名单是获知 Debian 重要消息的方法。此邮寄名单的流量极低 (每年只有十来个消息)，祗提供最重要的通知，诸如新版发布消息、选举新的计划领导人、或 Debian 年度会议。
→ https://lists.debian.org/debian-announce/
有关 Debian 的一般 (与定期) 消息经由 debian-news@lists.debian.org 邮寄名单发送。其流量尚可 (通常每月数则)，包括半定期的 'Debian 计划消息'，汇编计划内发生的小事。Debian 发展者在这里发布有必要让大众知悉的事，算是内幕消息。
→ https://lists.debian.org/debian-news/

社区 公关与媒体团队

Debian 的官方沟通管道由公关团队与媒体团队的志工担任。媒体团队是 Debian 计划领导人的代表发布官方的消息。公关团队较不正式且欢迎大家参与，提供稿件给 'Debian 计划消息' 或供稿给@debian Identi.ca 微部落客。
→ http://wiki.debian.org/Teams/Press
→ http://wiki.debian.org/Teams/Publicity
Debian 的演化与各团队的运作，参见 debian-devel-announce@lists.debian.org 邮寄名单。如其名称所示，其内容较适合发展者的口味，也适合关心稳定版以外版本发展的人。debian-announce@lists.debian.org 提供用户可见的结果，debian-devel-announce@lists.debian.org 则告知该等结果的产生方式。如专栏所示，'d-d-a' (如其名所示) 是 Debian 发展者必须订阅的邮寄名单。
→ https://lists.debian.org/debian-devel-announce/
详尽的非正式信息在 Debian 星球，汇集 Debian 奉献者在自身博客的文章。其内容不以 Debian 发展为限，还包括社区的活动与会员的动态。
→ http://planet.debian.org/
计划本身也是一种社区网络。Debian 本身只有以自由软件建构的官方平台 (如 Identi.ca 微博客平台，由 pump.io) 架构，Debian 的奉献者多半另有自己的推特帐号、脸书页面、Google+ 页面等。
→ https://identi.ca/debian
→ https://twitter.com/debian
→ https://www.facebook.com/debian
→ https://plus.google.com/111711190057359692089
