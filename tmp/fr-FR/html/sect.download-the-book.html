D.3. Télécharger le livre

Merci beaucoup pour votre don. Nous apprécions vraiment cela. Nous espérons que vous apprécierez notre livre !
Ce livre est disponible dans 3 format et en plusieurs langues (voir les autres ongles pour les traductions).
Le format PDF est le document correspondant au livre imprimé. Il ne convient pas pour la lecture sur de petits écrans. Le format EPUB est le format recommandé si vos appareils de lecture le supporte (la plupart des smartphones et de nombreux lecteurs e-book le supportent). Le format Mobipocket est principalement destiné aux lecteurs de livres électroniques qui ne prennent pas en charge EPUB (comme le Kindle).
Cliquez sur le lien de la version que vous souhaitez télécharger :
  • Anglais PDF (30 Mb)
  • Anglais EPUB (5 Mb)
  • Anglais Mobipocket (13 Mb)
