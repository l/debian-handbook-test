A.3. Linux Mint

Linux Mint is een (deels) gemeenschaps-onderhouden distributie, ondersteund door schenkingen en advertenties. Hun vlaggenschip product is gebaseerd om Ubuntu, maar ze bieden ook een “Linux Mint Debian Edition” variant die continu evolueert (omdat het is gebaseerd op Debian Testen). In beide gevallen houdt de initiële installatie het opstarten vanaf een LiveDVD in.
The distribution aims at simplifying access to advanced technologies, and provides specific graphical user interfaces on top of the usual software. For instance, Linux Mint relies on Cinnamon instead of GNOME by default (but it also includes MATE as well as Plasma and Xfce); similarly, the package management interface, although based on APT, provides a specific interface with an evaluation of the risk from each package update.
Linux Mint bevat een groot aantal gepatenteerde software om de gebruikerservaring, voor hen die deze nodig zouden hebben, te verbeteren. Bij voorbeeld: Adobe Flash en multimedia codecs.
