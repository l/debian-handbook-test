付録 C Debian 管理者ハンドブック

Debian Jessie との出会いから習熟まで

Debian GNU/Linux は大人気の非商用 Linux ディストリビューションであり、その信頼性と有効性で知られています。Debian プロジェクトは数千人の開発者からなる素晴らしいネットワークによって創造されメンテナンスされており、Debian 社会契約と強く結び付いています。Debian 社会契約という基本文書では、完全に自由なオペレーティングシステムでユーザのニーズを満足させるという Debian プロジェクトの目標が定義されています。そして Debian および派生ディストリビューション (特に Ubuntu) のエコシステムの成功により、Debian の技術に触れる管理者の数が増えつつあるのです。
本書 Debian 管理者ハンドブックは過去の 6 種類の版の成功を基盤にして Debian 8「Jessie」用に全体にわたって更新されました。本書は誰にとってもわかりやすく、実践的かつ自律的な Debian GNU/Linux 管理者になりたいと思っているすべての人に本質的要素を教えるものです。そして本書は有能な Linux 管理者が習得するべきすべての話題 (システムのインストールから更新、パッケージの作成とカーネルのコンパイル、監視、バックアップと移行) をカバーしています。もちろん熟練ユーザ向けの話題 (サービスを安全なものにする SELinux や AppArmor のセットアップ、自動インストール、Xen、KVM、LXC を使った仮想化) もカバーしています。
本書は本職のシステム管理者だけのために書かれたものではありません。自分のコンピュータで Debian または Ubuntu を使う人なら誰もが事実上の管理者であり、自分のシステムがどのように動いているのかに関して知ることは極めて価値のあることだと気が付くでしょう。問題を理解し解決する能力を身に付けることができれば、読者の皆様の非常に貴重な時間を節約することが可能になるでしょう。
Raphaël Hertzog はフランス国立応用科学院 (INSA) リヨン校を卒業したコンピュータサイエンスエンジニアであり、1997 年からずっと Debian 開発者として活動し続けています。彼はまた Debian GNU/Linux に特化したフランスの IT サービス企業 Freexian の創業者でもあり、Debian GNU/Linux ディストリビューションに対する最も傑出した貢献者の一人です。
Roland Mas はフランスのパリにあるテレコムブルターニュ電気通信国立大学 (ENST) を卒業した電気通信技術者です。彼は 2000 年から Debian 開発者であるのと同時に FusionForge ソフトウェアの開発者でもあります。その一方で、彼は Debian GNU/Linux システムのインストールと移行および共同作業ツールのセットアップに特化したフリーランスのコンサルタントとしても活躍しています。
本書には歴史があります。本書は最初フランス語の書籍 (Eyrolles から出版されている Cahier de l'Admin) として世に送り出され、資金調達に協力してくださった数百人のおかげで英語に翻訳されました。詳しい情報は https://debian-handbook.info に載っています。このウェブサイトでは本書の電子版を入手することも可能です。
