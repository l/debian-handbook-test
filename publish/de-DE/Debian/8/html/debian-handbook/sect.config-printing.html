8.7. Druckereinrichtung

Die Druckereinrichtung verursachte gewöhnlich sowohl für Administratoren als auch für Benutzer erhebliche Kopfschmerzen. Diese Kopfschmerzen gehören inzwischen dank cups, dem freien Druckserver, der das IPP-Protokoll (Internet Printing Protocol) verwendet, weitgehend der Vergangenheit an.
Dieses Programm ist über mehrere Debian-Pakete verteilt: cups ist der zentrale Druckserver; cups-bsd ist eine Kompatibilitätsschicht, die die Verwendung von Befehlen des traditionellen BSD-Drucksystems ermöglicht (den Daemon lpd, die Befehle lpr und lpq usw.); cups-client enthält eine Gruppe von Programmen, die mit dem Server zusammenarbeiten (einen Drucker sperren oder freigeben, gerade stattfindende Druckaufträge anzeigen oder löschen usw.); und schließlich enthält cups-driver-gutenprint eine Sammlung zusätzlicher Druckertreiber für cups.

GEMEINSCHAFT CUPS

CUPS (Common Unix Printing System) ist ein Projekt (und eine geschützte Handelsmarke) unter der Verwaltung von Apple.
→ http://www.easysw.com/
After installation of these different packages, cups is administered easily through a web interface accessible at the local address: http://localhost:631/. There you can add printers (including network printers), remove, and administer them. You can also administer cups with the graphical interface provided by the desktop environment. Finally, there is also the system-config-printer graphical interface (from the Debian package of the same name).

HINWEIS Veralterung von /etc/printcap

cups benutzt die jetzt veraltete Datei /etc/printcap nicht mehr. Programme, die sich auf diese Datei stützen, um eine Liste verfügbarer Drucker zu erhalten, versagen daher. Um dieses Problem zu vermeiden, sollten Sie diese Datei löschen und durch eine symbolische Verknüpfung (siehe ZURÜCK ZU DEN GRUNDLAGEN Symbolische Verknüpfungen in der Seitenleiste) zur Datei /var/run/cups/printcap ersetzen, die von cups beibehalten wird, um Kompatibilität zu gewährleisten.
