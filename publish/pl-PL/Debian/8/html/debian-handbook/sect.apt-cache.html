6.3. Komenda apt-cache

The apt-cache command can display much of the information stored in APT's internal database. This information is a sort of cache since it is gathered from the different sources listed in the sources.list file. This happens during the apt update operation.

SŁOWNIK Pamięć podręczna

Pamięć podręczna jest rodzajem tymczasowego systemu magazynowania używanym w celu przyspieszenia dostępu do często używanych danych, podczas gdy zwykłe sposoby dostępu są kosztowne (wydajnościowo). Ten sposób może być stosowany w wielu sytuacjach i na różną skalę, od rdzeni mikroprocesorów do zaawansowanych systemów magazynowania.
W przypadku APTa, referencyjne pliki Packages to te znajdujące się na serwerach lustrzanych Debiana. W związku z tym byłoby bardzo nieefektywne, aby przy każdym wyszukiwaniu wykorzystywać sieć do przeszukiwania bazy danych dostępnych pakietów. Właśnie dlatego APT przechowuje kopię tych plików (w /var/lib/apt/lists/) i wyszukiwania są wykonywane na tych lokalnych plikach. Podobnie, /var/cache/apt/archives/ zawiera pamięć podręczną już pobranych pakietów, aby uniknąć ich ponownego pobierania w celu reinstalacji po usunięciu.
The apt-cache command can do keyword-based package searches with apt-cache search keyword. It can also display the headers of the package's available versions with apt-cache show package. This command provides the package's description, its dependencies, the name of its maintainer, etc. Note that apt search, apt show, aptitude search, aptitude show work in the same way.

ALTERNATYWA axi-cache

apt-cache search jest bardzo prostym narzędziem, właściwie stosuje komendę grep dla opisu pakietu. Często zwraca zbyt wiele wyników albo wcale, jeżeli zostanie zawarte zbyt wiele słów kluczowych.
Z drugiej strony axi-cache search określenie dostarcza lepsze wyniki, posortowane względem trafności. Używa silnika wyszukiwania Xapian i jest częścią pakietu apt-xapian-index, który indeksuje wszystkie informacje pakietu (i nawet więcej, jak pliki .desktop ze wszystkich pakietów Debiana). Rozróżnia także znaczniki (spójrz na fiszkę IDĄC DALEJ Pole Tag) i zwraca wyniki w ciągu milisekund. 
$ axi-cache search package use::searching
105 results found.
Results 1-20:

98% debtags - Enables support for package tags
94% debian-goodies - Small toolbox-style utilities 
93% dpkg-awk - Gawk script to parse /var/lib/dpkg/{status,available} and Packages

`axi-cache more' will give more results
Niektóre funkcje są rzadziej używane. Na przykład apt-cache policy wyświetla zarówno priorytety źródeł pakietów i pojedynczych pakietów. Kolejnym przykładem jest apt-cache dumpavail, która wyświetla nagłówki wszystkich dostępnych wersji wszystkich pakietów. apt-cache pkgnames wyświetla listę wszystkich pakietów, które choć raz pojawiają się w pamięci podręcznej.
