8.7. پیکربندی چاپگر

پیکربندی چاپگر سابق بر این سردردهای بسیار شدیدی را برای مدیرسیستم‌ها بوجود می‌آورد. اکنون این دشواری‌ها حل شده‌اند، به لطف cups، سرور چاپ رایگانی که از پروتکل IPP یا Internet Printing Protocol استفاده می‌کند.
This program is divided over several Debian packages: cups is the central print server; cups-bsd is a compatibility layer allowing use of commands from the traditional BSD printing system (lpd daemon, lpr and lpq commands, etc.); cups-client contains a group of programs to interact with the server (block or unblock a printer, view or delete print jobs in progress, etc.); and finally, printer-driver-gutenprint contains a collection of additional printer drivers for cups.

جامعه کاربری CUPS

سیستم چاپ متداول یونیکس یا CUPS پروژه (و نام تجاری) است که توسط شرکت اپل مدیریت می‌شود.
→ http://www.cups.org/
پس از نصب این بسته‌های مختلف، cups به راحتی از طریق یک رابط وب از نشانی محلی http://localhost:631/ مدیریت می‌شود. آنجا می‌توانید چاپگرهای خود را اضافه (از جمله چاپگرهای شبکه) آن‌ها را حذف و مدیریت کنید. همچنین می‌توانید با استفاده از رابط گرافیکی میزکار خود نیز با cups ارتباط برقرار کنید. در نهایت، همچنین رابط گرافیکی system-config-printer در بسته‌ای با همین نام به منظور مدیریت چاپگرها وجود دارد.

یادداشت از رده خارج شدن /etc/printcap

cups no longer uses the /etc/printcap file, which is now obsolete. Programs that rely upon this file to get a list of available printers will, thus, fail. To avoid this problem, delete this file and make it a symbolic link (see sidebar بازگشت به مقدمات پیوندهای نمادین) to /run/cups/printcap, which is maintained by cups to ensure compatibility.
