4.3. 首次啟動之後

沒有任何明確的選擇 (或使用 “GNOME” 選項) 就啟動 “Debian 桌面環境”，電腦會顯示 gdm3 登入管理員。

圖形 4.14. 首次啟動

已建立的使用者可以登錄並立即開始工作。

4.3.1. 安裝其他軟體

安裝的套件符合當初選定的，但不見得符合實際的需要。在此情況下，需使用套件管理工具微調選定的安裝套件。最常用的工具 (已安裝於 “Debian 桌面環境”) 是 apt (從命令進入) 與 synaptic (位於選單內的 “Synaptic 套件管理器”)。
為了保法安裝程式的一致性，Debian 新增特定用途的 “任務” (郵件伺服器、檔案伺服器等)。在安裝過程中曾經選擇過，現在還可利用套件管理工具 aptitude (其任務列在不同的區塊) 與 synaptic (經由選單 Edit → Mark Packages by Task…)等，再選擇一次。
Aptitude 是 APT 的全螢幕文字模式。允許使用者依類別瀏覽可用的套件清單 (已安裝或待安裝套件，依任務或區塊分類)，並檢視相關的資訊 (相依性、衝突、說明等)。每個套件可標示為 “安裝” (可以安裝，+ 鍵) 或 “移除” (可移除，- 鍵)。按. All of these operations will be conducted simultaneously once you've confirmed them by pressing the g 鍵(“g” 表示 “go!”) 確認後就會同時執行該等作業。若您忘了曾安裝的套件，別擔心；祗要在安裝完成後再執行 aptitude 命令即可。

技巧 Debian 為非英語系使用者著想

若干任務專門用於在地化系統為英語以外的語系。包括翻譯文件、字典、與供其他語系使用的套件。安裝過程已選擇非英語的語系，則自動執行適當的任務。

文化 dselect，安裝套件的舊介面

aptitude 問世前，標準的選擇套件為 dselect，採用 dpkg 安裝的舊介面。對初學者有點難度，不建議使用它。
Of course, it is possible not to select any task to be installed. In this case, you can manually install the desired software with the apt or aptitude command (which are both accessible from the command line).

術語 套件相依性、衝突

在 Debian 的套件管理中，“相依性” 係指另個套件中的某個功能恰是滿足本套件所需。反過來說，“衝突” 係指不能同時併存的兩個套件。
這些概念的詳情在 章 5, 套件系統：工具與基本原則。

4.3.2. 系統升級

A first apt upgrade (a command used to automatically update installed programs) is generally required, especially for possible security updates issued since the release of the latest Debian stable version. These updates may involve some additional questions through debconf, the standard Debian configuration tool. For further information on these updates conducted by apt, please refer to 節 6.2.3, “系統升級”.
