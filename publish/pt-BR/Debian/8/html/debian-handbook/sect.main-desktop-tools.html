13.4.1. Evolution

COMUNIDADE Pacotes populares

Instalar o pacote popularity-contest habilita a participação em uma pesquisa automatizada que informa ao projeto Debian os pacotes mais populares. Um script é rodado semanalmente pelo cron que envia (por HTTP ou email) uma lista anônima dos pacotes instalados e a última data de acesso aos arquivos que eles contém. Isso permite diferenciar, dentre os pacotes instalados, aqueles que são realmente usados.
Essa informação é uma grande ajuda ao projeto Debian. Ela é usada para determinar quais pacotes devem estar nos primeiros discos de instalação. Dados de instalação também são um fator importante usado para decidir quando remover um pacote com muito poucos usuários na distribuição. Nós cordialmente recomendamos a instalação do pacote popularity-contest e a participação na pesquisa.
Os dados coletados tornam-se públicos todos os dias.
→ http://popcon.debian.org/
Essas estatísticas também podem ajudar a escolher entre dois pacotes que parecem equivalentes. Escolher o pacote mais popular aumenta a probabilidade de fazer uma boa escolha.
O Evolution é o cliente de email do GNOME e pode ser instalado com apt-get install evolution. O Evolution vai além de um simples cliente de email, ele também fornece um calendário, um livro de endereço, uma lista de tarefas, e uma aplicação de memo (nota de forma livre). Seus componentes de email incluem um poderoso sistema de indexação de mensagens, e permite a criação de pastas virtuais baseado em consultas de pesquisa em todas as mensagens arquivadas. Em outras palavras, todas as mensagens são armazenadas da mesma forma, mas exibidas em uma organização baseada em pastas, sendo cada pasta contendo mensagens que coincidem com um critério de conjunto de filtros.

Figura 13.4. O programa de e-mail Evolution

Uma extensão para o Evolution permite a integração com o sistema de email Microsoft Exchange; o pacote necessário é o evolution-ews.

13.4.2. KMail

O software de email do KDE pode ser instalado com apt-get install kmail. O KMail apenas lida com email, mas ele pertence a um conjunto de softwares chamado KDE-PIM (de Personal Information Manager - Gerenciador de Informações Pessoais) que inclui recursos como livro de endereço, um componente de calendário, e assim por diante. O KMail tem todos os recursos que se espera de um excelente cliente de email.

13.4.3. Thunderbird e Icedove

Esse software de email, incluído no pacote icedove, é parte da suíte de software Mozilla. Vários conjuntos de regionalização estão disponíveis no pacotes icedove-l10n-*; a extensão enigmail lida com criptografia e assinatura de mensagem (entretanto, não está disponível em todas as linguas).

Figura 13.6. O programa de e-mail Icedove

O Thunderbird é um dos melhores clientes de email, e parece ser um grande sucesso, como o Mozilla Firefox.
Falando claramente, o Debian Jessie contém o Icedove, e não o Thunderbird, por questões legais que nós iremos detalhar na barra lateral CULTURA Iceweasel, Firefox e outros; mas fora seus nomes (e icones), não existe diferença real entre eles.
